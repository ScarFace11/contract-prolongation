# Contract Prolongation Analysis

Анализ коэффициентов пролонгации договоров аккаунт-менеджеров за 2023 год.

## Файлы

- `prolongation_analysis.py` — основной скрипт расчёта KPI пролонгаций
- `prolongation_report.xlsx` — итоговый отчёт в Excel (3 листа: Department, Managers, Year KPI)
- `prolongation_chart.png` — график динамики коэффициентов по месяцам
- `attached_assets/` — исходные данные (prolongations.csv, financial_data.csv, ТЗ)

## User preferences

- Все файлы пушить на GitHub в репозиторий `ScarFace11/contract-prolongation`
